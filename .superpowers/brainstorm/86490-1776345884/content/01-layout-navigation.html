Key Design Decisions

  • Left sidebar: "MCP Servers" sits alongside Models, Experiments, Deployments, Prompts — showing MCPs as a peer asset type in MLflow
  • Feature toggle: Top-right pill toggle switches between "3.5 DP" and "3.next" — unobtrusive but always accessible
  • Purple accent (#7B3FE4): MLflow's signature color for active states and primary actions
  • Breadcrumb nav: MLflow → MCP Servers → [Server Name] → [Version] for drill-down context
  • "Register MCP Server": Primary action button, parallels MLflow's "Create Model" pattern
