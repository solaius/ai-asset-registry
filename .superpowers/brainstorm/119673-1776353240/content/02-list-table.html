Table Design Decisions

  • Columns follow MLflow's model registry pattern: Name (with icon + link), Latest Version, Lifecycle State, then MCP-specific columns (Source, Tools count, Tags)
  • Name column — brand-colored icon + clickable link (matches MLflow model names)
  • Lifecycle State badges — DuBois Tag component colors: Published (lime), Draft (default), Deprecated (lemon)
  • Source column — Red Hat / Partner / Community (MCP-specific, no MLflow equivalent)
  • Tools column — count of tools exposed by the MCP server (MCP-specific)
  • Row hover — subtle background tint matching DuBois table hover
  • Data variety — most Published, one Draft (Terraform), one Deprecated (MariaDB) to show state diversity

3.next Additions (when toggle is ON)

These columns/badges would appear when the 3.next feature toggle is enabled:

  • Approval Status column — Pending / Approved / Rejected badges
  • Verification badge on name — checkmark for verified sources
  • Certification badge — "Certified" tag for compliance-approved MCPs
